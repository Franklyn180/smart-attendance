# Smart Attendance Management System

A full-stack attendance management application built with React + Vite frontend, Python backend, and MySQL database.

Features:
- Student registration
- QR-code attendance tracking
- Lecturer dashboard
- Attendance analytics
- Course management

## Project layout

- `frontend/` — Vite-powered React interface
- `backend/` — FastAPI backend with MySQL database support
- `.gitignore` — excludes local environment files and dependencies
- `.dockerignore` — lightens container build context by excluding node_modules, Python caches, and environment files
